# Reviews Index

本目录存放阶段性审计、接口评审和项目汇总文档。

## 放置规则

适合放在这里的文档：

- `*_audit_YYYY-MM-DD.md`
- `*_summary_YYYY-MM-DD.md`
- 接口迁移建议
- 阶段汇总结论

这些文档通常带日期，反映某一阶段的判断，不等同于长期稳定规范。

说明：

- 评审文档允许保留当时的命令、路径和上下文快照
- 当前正式规则应以 `README.md`、`docs/project_conventions.md`、`docs/system_*.md` 为准
