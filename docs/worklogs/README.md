# Worklogs Index

本目录存放按日期记录的开发过程日志。

命名规则：

- `worklog_YYYY-MM-DD.md`

放置原则：

- 记录过程、决策、当日结论
- 不作为长期稳定规范文档
- 需要长期引用的结论，应再提炼到 `docs/` 根目录或 `reports/benchmarks/`

说明：

- 工作日志允许保留当时的命令和局部上下文
- 当前正式规范应以根 README 和 `docs/` 根目录稳定文档为准
